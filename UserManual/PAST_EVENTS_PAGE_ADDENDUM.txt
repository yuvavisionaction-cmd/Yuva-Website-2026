================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: Badge not showing gradient
──────────────────────────────────
SOLUTION:
1. Check CSS variables are defined in :root
2. Verify --gradient-green is set correctly
3. Clear browser cache
4. Check for CSS conflicts

ISSUE: Card not hovering properly
──────────────────────────────────
SOLUTION:
1. Verify .latest-event-card class is applied
2. Check for z-index conflicts
3. Ensure no parent overflow:hidden
4. Test in different browsers

ISSUE: Image not zooming on hover
──────────────────────────────────
SOLUTION:
1. Check .event-image-wrapper has overflow:hidden
2. Verify img has transition property
3. Test hover state in DevTools
4. Check for JavaScript conflicts

ISSUE: Responsive layout not working
─────────────────────────────────────
SOLUTION:
1. Verify viewport meta tag in HTML
2. Check media query breakpoints
3. Test in responsive mode
4. Clear browser cache
